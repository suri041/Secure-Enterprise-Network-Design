# 🔐 Secure Enterprise Network Design

> A comprehensive enterprise network infrastructure designed and configured using **Cisco Packet Tracer**, implementing secure network architecture, dynamic routing, firewall security, wireless networking, VoIP services, and enterprise security best practices.

---

## 📌 Project Overview

This project demonstrates the design and implementation of a secure enterprise network based on a hierarchical architecture. The network provides secure communication between departments while ensuring scalability, reliability, and high availability.

The infrastructure integrates routing, switching, security, wireless networking, and VoIP technologies commonly used in enterprise environments.

---

## 🎯 Project Objectives

- Design a scalable enterprise network
- Implement secure VLAN segmentation
- Configure dynamic routing using OSPF
- Deploy Cisco ASA Firewall
- Secure the network using ACLs and Port Security
- Implement NAT for external connectivity
- Configure Wireless LAN Controller (WLC)
- Deploy Wireless Access Points
- Configure VoIP services
- Verify end-to-end connectivity

---

# 🏗️ Network Architecture

The enterprise network follows the Cisco hierarchical design model.

- Core Layer
- Cisco ASA Firewall
- Distribution Layer
- Access Layer
- Edge Router
- Wireless LAN Controller (WLC)
- Wireless Access Point
- VoIP Phones
- End Devices
- Servers
